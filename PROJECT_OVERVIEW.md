# 项目概览

## 🎯 项目简介

这是一个专为比特币投资者设计的投资日记应用，帮助你记录每一笔交易、追踪持仓收益、管理投资策略。

## ✨ 核心特性

### 📝 交易管理
- 记录买入/卖出交易
- 包含日期、价格、数量、备注
- 为每笔交易添加投资思考

### 💰 实时追踪
- 自动获取比特币实时价格（Coinbase API）
- 每分钟自动更新
- 显示更新时间

### 📊 统计分析
- **持仓数量**：实时显示当前持有的BTC总量
- **平均成本**：自动计算平均买入价格
- **总投入**：净投入金额（买入-卖出）
- **当前市值**：按实时价格计算
- **账面收益**：显示盈亏金额和收益率百分比

### 💾 数据管理
- 导出为JSON格式
- 支持导入恢复
- 本地存储（localStorage）
- 包含示例数据

### 🔐 安全保护
- 密码验证机制
- 支持URL参数传递密码
- Session状态管理
- 数据仅存储在本地

### 🎨 界面设计
- 简约现代的设计风格
- 渐变背景
- 卡片式布局
- 响应式设计（支持手机/平板）
- 流畅的动画效果
- 直观的颜色标识

## 🔧 核心功能实现

### 价格更新
```javascript
// 每分钟从Coinbase API获取最新价格
setInterval(updateBTCPrice, 60000);
```

### 收益计算
```
持仓 = Σ买入数量 - Σ卖出数量
成本 = Σ买入金额 ÷ Σ买入数量
市值 = 持仓 × 当前价格
收益 = 市值 - (Σ买入金额 - Σ卖出金额)
收益率 = 收益 ÷ 净投入 × 100%
```

### 数据存储
```javascript
// 存储到localStorage
localStorage.setItem('btc_transactions', JSON.stringify(data));

// 读取数据
const data = JSON.parse(localStorage.getItem('btc_transactions'));
```
