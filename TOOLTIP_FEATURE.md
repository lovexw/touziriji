# 收益指标 Tooltip 说明功能

## 功能概述

为三个收益指标添加了鼠标悬停提示（Tooltip）功能，帮助用户更好地理解不同收益类型的含义和特点。

## 修改内容

### 1. HTML 修改 (index.html)

为三个收益指标添加了带提示的问号图标：
- **账面收益（未实现）**
- **已实现收益**
- **总收益**

每个指标都包含：
- 橙色圆形问号图标
- 鼠标悬停时显示的详细解释
- 清晰的标题和特点说明

### 2. CSS 修改 (styles.css)

添加了完整的 tooltip 样式系统：

#### 核心样式
- `.tooltip-icon` - 问号图标容器
- `.tooltip-icon .icon` - 橙色圆形问号
- `.tooltip-content` - 提示内容浮窗
- `.tooltip-content::after` - 指向箭头

#### 交互效果
- 鼠标悬停时问号变深色并放大 (scale 1.1)
- 提示框平滑淡入淡出动画
- 深色渐变背景提高可读性
- 指向箭头清晰指示来源

## 收益指标说明

### 账面收益（未实现）
**定义：** 当前持有的 BTC 还没卖出，暂时产生的盈利/亏损  
**特点：** 随 BTC 价格实时变动，属于"浮动收益"（没实际落袋）

### 已实现收益
**定义：** 之前卖出部分 BTC 后，实际拿到手的盈利  
**特点：** 收益已经确定，不会再随价格变化

### 总收益
**定义：** 账面收益（未实现） + 已实现收益 的总和  
**特点：** 反映这笔 BTC 投资的"整体盈利情况"

## 技术实现

### Tooltip 定位
```css
position: absolute;
bottom: calc(100% + 8px);  /* 向上偏移 */
left: 50%;
transform: translateX(-50%); /* 水平居中 */
```

### 悬停动画
```css
opacity: 0;
visibility: hidden;
transition: all 0.3s ease;
```

### 箭头指示
```css
.tooltip-content::after {
    border: 6px solid transparent;
    border-top-color: #374151; /* 与背景渐变底色一致 */
}
```
